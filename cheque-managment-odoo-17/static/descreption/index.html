Cheque Management in Accounts

Cheque management will help the life cycle for the cheque in company, which will help you to track the cheques received and given to suppliers and so on... , with this module you can track the cheques are cleared or bounced or not paid, in this module we have complete workflow to handle the cheques.

This module will provide with below features

  • Incoming cheques
  • Outgoing cheques
  • Track the dates when the cheque is cleared
  • Track if the cheque is bounced and send back
  • Cheques returned once bounced
  • Track cheque deposited to bank
  • You can attached the cheque copies

In invoicing Cheque Management menu can be visible with incoming and outgoing cheques.

Will have one flow of cheque management, create incoming cheque by entering all necessary data and the status would be in Draft.

Warning message to avoid duplication of cheque number.

Once cheque is registered there would be four buttons visible for further steps, now will check with Cashed flow once clicked it will open wizard to register the entries







Will have Bounced cheque flow once bounced it will post entries.




Once cheque is returned journal entries get canceled for that you need allow to cancel journal entries




Once cheque is deposited can be tracked in system.

It can be cancelled




Attachment can be made within the form like cheque copies, deposit slips etc..
